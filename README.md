# Python Tkinter Calculator

A beginner-friendly calculator application built using Python and Tkinter.

## Features
- Basic arithmetic operations (+, -, *, /)
- Clear (C) button
- Keyboard input support
- Error handling (invalid input, division by zero)
- Simple and clean GUI

## Tech Stack
- Python
- Tkinter

## What I Learned
- Building GUIs using Tkinter
- Handling button and keyboard events
- Basic state management in applications
- Improving user experience through logic fixes

## Note
This project was created as a learning exercise to understand Python GUI development.
